🍱 MessEase - Didi's Mess Management System
MessEase ek custom-built MERN stack application hai jo hyper-local mess services (Didi's Mess) ko manage karne ke liye banaya gaya hai. Yeh system purane paper-based "Khata" system ko digital platform par lata hai, jisse attendance, billing aur expenses manage karna aasan ho jata hai.

🚀 Features
👩‍🍳 Admin Dashboard (For Didi)
Secure Access: Master PIN-protected login (9988).

Student Management: Naye students ko register karna aur unki details (Address, Emergency Contact) save karna.

Attendance Tracking: Rozana ka Breakfast, Lunch aur Dinner mark karne ka asan interface.

Smart Billing: Attendance ke basis par automatic bill calculation.

Expense Tracker: Mahine ke kharche (sabzi, rashan, rent) manage karna aur Net Profit dekhna.

Automated Alerts: 30 din se zyada purane bills par WhatsApp aur Email reminders bhejna.

PDF Receipts: Students ke liye professional payment receipts generate karna.

👨‍🎓 Student Portal
Personal Login: Phone number aur unique PIN ke saath login.

Live Bill: Apna current due amount aur payment status dekhna.

Attendance History: Puraane attendance records (B/L/D tags) check karna.

Today's Menu: Aaj khane mein kya bana hai, wo live dekhna.

Profile Update: Apna address aur emergency contact details khud update karna.

🛠️ Tech Stack
Frontend: React.js, Vite, Lucide Icons

Backend: Node.js, Express.js

Database: MongoDB Atlas (Mongoose)

Security: PIN-based Authentication, Environment Variables (.env)

Utilities: jsPDF (Receipts), Node-Cron (Automated tasks), Nodemailer (Emails)
